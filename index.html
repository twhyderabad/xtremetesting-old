About

ThoughtWorks Hyderabad is glad to announce a free-for-all Extreme Programming for developers camp. It is a 4-day course with coding and hands on exercises, designed to build a community of developers who practice extreme programming (XP) methodologies in software development life cycle. This course will help developers gain hands-on experience on XP methodologies which can be applied to build better softwares.

Topics that will be covered

  • Pair Programming
  • Unit Testing (Automated Testing)
  • Test Driven Development
  • Clean Code & Refactoring
  • Continuous Integration and deployment
  • Object oriented programming with some bits of Design Patterns
  • Along the way you will also learn Standups, Retrospectives and other Agile practices.

Who can apply?

The Xtreme Programming concepts are not related to any programming languages. But some of the concepts and hands-on are explained using examples in JAVA, basic understanding of JAVA is more than enough to follow the workshop. You can implement the examples in any language of your choice.

It is NOT restricted to only Hyderabadis. If you can manage your own travel and stay, you are most welcome to apply.

What is the selection Criteria?

Selection will be based on assessment of your code for a coding problem. There are two problems in the link given below, you can solve any ONE of them.

You can use any programming language of your liking to solve it. By design, the difficulty level of the problem will be low. It will test your basic coding skills and commitment to finish a given task (hopefully, implying the commitment to finish the course).

You can choose any object oriented language to solve the problem.

Why free?

Well, we have been active evangelists of these development practices for many years. Today we are very happy to see that there is increased interest and awareness of extreme programming concepts across India and in Hyderabad as well. However, we feel that there is a big gap in the understanding and appreciation of these practices. We would like to share our knowledge to build community around these practices.
